# Scroll Capture Extension

這是一款專為截取「內嵌可滾動區塊」而設計的 Chrome 擴充功能，能夠自動偵測頁面中最大的 scrollable 區塊，並完整截圖下載。適用於 momo、蝦皮、LINE 購物等電商頁面中嵌入式內容。

## 📂 專案結構

- `src/`：原始擴充功能程式碼
- `assets/`：圖片與示意圖
- `README.md`：說明文件
- `LICENSE`：MIT 授權
- `.gitignore`：Git 忽略設定

## 🛠 安裝方式

1. 前往 Chrome：`chrome://extensions`
2. 開啟開發人員模式
3. 點「載入未封裝項目」→ 選擇 `src/` 資料夾
